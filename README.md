# TailwindCSS Starter Project
This is a tailwindcss starter project template for static website including live-server.

## Packages
- tailwindcss `3.3.1`
- live-server `1.2.2`

## CDN(s)
- Fontawrsome (cdnjs) `6.4.0`

## How use
### 1. To start liver server
```
npm run serve
```

### 2. To compile tailwindcss
```
npm run compile
```
